# Blog REST API

## Project Overview
This project is a Blog REST API developed using Node.js and Express.js. It performs CRUD operations for blog posts.

## Technologies Used
- Node.js
- Express.js
- JavaScript
- JSON

## API Endpoints

POST /posts - Create a blog post

GET /posts - Get all blog posts

GET /posts/:id - Get a single blog post

PUT /posts/:id - Update a blog post

DELETE /posts/:id - Delete a blog post

## Installation

1. Clone the repository
2. Run npm install
3. Run node app.js

## Testing

Use Postman to test all the API endpoints.
